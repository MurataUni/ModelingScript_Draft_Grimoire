グリモア
====

## Description

「Gのレコンギスタ」からグリモアの下書きモデリング  
ZBrushCoreにてモデリングを行う際のアタリを取る用に作成した下書き  

このスクリプトはファンアートです。  
原作者様・出版社様とは一切関係ありません。

* modeling.py - 3Dモデルを出力するためのスクリプト

## Requirement

Python 3.7で動作確認  
[Harbor3d](https://github.com/MarataUni/Harbor3d)がインストールされている環境で実行できる

## Usage

1. modeling.pyを実行
